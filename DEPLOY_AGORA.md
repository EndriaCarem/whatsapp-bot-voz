# 🚀 FAÇA O DEPLOY AGORA!

## 3 Passos para sua interface 3D estar online:

---

### ✅ PASSO 1: Gerar Token Vercel

1. Abra: https://vercel.com/account/tokens
2. Clique "Create" (Token novo)
3. Nome: `axolotl-deploy`
4. **Copie o token inteiro**

---

### ✅ PASSO 2: Conectar GitHub na Vercel

1. Abra: https://vercel.com/new
2. Clique "Continue with GitHub"
3. Selecione: `EndriaCarem/whatsapp-bot-voz`
4. Clique "Import"
5. **Pronto!** Vercel vai fazer o deploy automaticamente

Você vai receber uma URL como:
```
https://whatsapp-bot-voz-xyz.vercel.app/advanced.html
```

---

### ✅ PASSO 3 (Opcional): Deploy via CLI local

Se quiser fazer pelo terminal:

```bash
# Instalar Vercel CLI
npm install -g vercel@latest

# Fazer login
vercel login

# Deploy
./deploy.sh
```
